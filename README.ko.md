# Switch 병합 / 분할 도구 (GUI)

- Nintendo Switch NSP, NSZ, XCI, XCZ 파일을 병합 및 분할할 수 있는 경량 GUI 도구
- 디스크에 임시 파일을 생성하지 않는 스트리밍 파이프라인 방식으로 처리

## 주요 기능

### 분할된 컨테이너 파일 병합

- 분할된 컨테이너 파일을 하나의 파일로 결합
- 드래그 앤 드롭 지원
- 자동 포맷 감지

### 컨테이너 파일 분할

- 대용량 컨테이너 파일을 작은 파트로 분할

---

## 지원 포맷

- NSP (병합 / 분할)
- XCI (병합 / 분할)
- NSZ (병합 / 분할, 자동 압축 해제)
- XCZ (병합 / 분할, 자동 압축 해제)

모든 작업 결과는 NSP 파일로 출력됩니다.

---

## 자동 압축 해제

압축 포맷 사용 시:

- NSZ 파일 자동 압축 해제
- XCZ 파일 자동 압축 해제

압축 해제는 처리 과정 중 실시간으로 수행됩니다.

---

## 스트리밍 처리 방식

이 도구는 스트리밍 파이프라인을 사용하여:

- 병합 및 분할 과정 중 데이터를 실시간 처리
- 임시 파일 생성 방지
- 디스크 사용량 감소
- 성능 향상

### 기존 방식

1. NSZ/XCZ 압축 해제
2. 임시 NSP 생성
3. 병합 또는 분할 수행

이 도구는 모든 과정을 **한 번에 처리**합니다.

---

## 사용 사례

- 분할된 NSP/XCI 파일 작업
- 압축된 NSZ/XCZ 컨테이너 처리
- 매우 큰 파일 처리
- 자동화 워크플로우 구성

## 개발 환경

- Visual Studio 2022 — 주요 개발 환경
- .NET 8.0 (LTS) — 고성능 크로스 플랫폼 런타임
- LibHac — Nintendo Switch 파일 시스템 및 컨테이너 처리
- WPF — Windows 데스크톱 GUI 프레임워크
